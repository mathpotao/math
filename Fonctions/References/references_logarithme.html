La fonction logarithme

  • Notation : La fonction logarithme népérien est définie sur $] 0 ; + \infty[$ et on note : $f(x)=\text{ln}(x)$.

    Propriétés : La fonction ln est strictement strictement croissante sur $] 0 ; + \infty[$.

    Courbe :

    Limites :
    $\displaystyle{\lim_{x \to 0} \left( \text{ln}(x) \right) = - \infty}$
    $\displaystyle{\lim_{x \to + \infty} \left( \text{ln}(x) \right) = + \infty }$

    Exemple 1 : Soit $f_1(x)=5-\text{ln}(x)$.
    Par soustraction : $\displaystyle{\lim_{x \to 0} f_1(x) = + \infty}$.

    Exemple 2 : Soit $f_2(x)=\text{ln}(x)+1$.
    Par somme : $\displaystyle{\lim_{x \to + \infty} f_2(x) = + \infty}$.

    Dérivée : $f$ est dérivable sur $] 0 ; + \infty[$ et :
    $f'(x)=\dfrac{1}{x}$.

    Exemple 3 : Soit $T_1$ la tangente à la courbe de la fonction ln au point d'abscisse $1$ et $T_2$ la tangente à la courbe de la fonction ln au point d'abscisse $2$.
    Le coefficient directeur de $T_1$ est $f'(1)=\dfrac{1}{1} = 1$.
    Le coefficient directeur de $T_2$ est $f'(2)=\dfrac{1}{2}$.
