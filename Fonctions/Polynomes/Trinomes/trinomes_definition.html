Définition

  • Définition : Un trinôme est un polynôme de degré $2$ :
    $f(x)=ax^2+bx+c$.

    Exemple 1 : $f(x)=x^2-2x+3$.

    Ecriture d'un trinôme : Un trinôme peut s'écrire sous trois formes différentes :
    - développée : $f(x) = ax^2+bx+c$ ;
    - canonique : $f(x) = a(x- \alpha)^2+\beta$ ;
    - factorisée : $f(x) = a(x-x_1)(x-x_2)$.

    Exemple 2 :
    - forme développée : $f(x)=x^2-4x+3$;
    - forme canonique : $f(x)=(x-2)^2+1$;
    - forme factorisée : $f(x)=(x-1)(x-3)$.
