Exercice 1

  • 1. Soit $f(x) = x^3+x^2$.

     
    On a :
    • 1 : $f''(x) = 3x^2+2x$
    • 2 : $f''(x) = 6x+2$
    • 3 : $f''(x) = 6x$
    • 4 : $f''(x) = 3x^2$

    choisir le numéro de la bonne réponse :
     
    On a :
    • 1 : $f$ est convexe sur $]- \infty ; 3]$
    • 2 : $f$ est convexe sur $]- 3 ; + \infty]$
    • 3 : $f$ est convexe sur $]- \infty ; + \infty]$
    • 4 : $f$ est convexe sur $]3 ; + \infty]$

    2. Soit $g(x) = 2 \text{ln}(x)+x^2$.

     
    On a :
    • 1 : $g''(x) = \dfrac{2x^2+2}{x}$
    • 2 : $g''(x) = 2 \text{ln}(x)+2x$
    • 3 : $g''(x) = \dfrac{1}{x}+2x$
    • 4 : $g''(x) = \dfrac{-2+2x^2}{x^2}$

    choisir le numéro de la bonne réponse :
     
    On a :
    • 1 : $g$ est convexe sur $]0;1]$
    • 2 : $g$ est convexe sur $[1 ; + \infty]$
    • 3 : $g$ est convexe sur $]0;+ \infty[$
    • 4 : $g$ est convexe sur $[2 ; + \infty]$
