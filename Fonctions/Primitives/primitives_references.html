Primitives des fonctions de référence

  • Fonction carrée : $f(x)=x^2$ :
    une primitive de $f$ est définie sur $\mathbb{R}$ par :
    $F(x) = \dfrac{1}{3} x^3$.

    Fonction cube : $f(x)=x^3$ :
    une primitive de $f$ est définie sur $\mathbb{R}$ par :
    $F(x) = \dfrac{1}{4} x^4$.

    Fonction puissance : $f(x)=x^n$ :
    une primitive de $f$ est définie sur $\mathbb{R}$ par :
    $F(x) = \dfrac{1}{n+1} x^{n+1}$.

    Fonction inverse : $f(x)=\dfrac{1}{x}$ :
    une primitive de $f$ est définie sur $]0 ; + \infty[$ par :
    $F(x) = \text{ln}(x)$.

    Inverse de la racine carrée : $f(x)= \dfrac{1}{\sqrt{x}}$ :
    une primitive de $f$ est définie sur $]0 ; + \infty[$ par :
    $F(x) = 2 \sqrt{x}$.

    Fonction exponentielle : $f(x)= \text{e}^x$ :
    une primitive de $f$ est définie sur $\mathbb{R}$ par :
    $F(x) = \text{e}^x$.

    Fonction logarithme : $f(x)= \text{ln}(x)$ :
    une primitive de $f$ est définie sur $]0 ; + \infty[$ par :
    $F(x) = x \text{ln}(x)-x$.

    Fonction cosinus : $f(x)= \text{cos}(x)$ :
    une primitive de $f$ est définie sur $\mathbb{R}$ par :
    $F(x) = \text{sin}(x)$.

    Fonction sinus : $f(x)= \text{sin}(x)$ :
    une primitive de $f$ est définie sur $\mathbb{R}$ par :
    $F(x) = -\text{cos}(x)$.

    Exemple 1 : Soit $f(x)=x^5$.
    une primitive de $f$ est définie sur $\mathbb{R}$ par :
    $F(x) = \dfrac{1}{6} x^6$.
