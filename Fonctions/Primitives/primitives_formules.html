Formules sur les primitives

  • Somme :
    Soit $F$ une primitive de $f$ et $G$ une primitive de $g$.
    Une primitive de $f+g$ est : $F+G$.
    $(F+G)' = f+g$.

    Exemple 1 : Soit $f(x)=\text{e}^x+2x$.
    Une primitive de $f$ est :
    $F(x) = \text{e}^x + x^2$.

    Produit par un réel :
    Soit $k$ un réel et $F$ une primitive de $f$.
    Une primitve de $kf$ est : $kF$.
    $(kF)' = kf$.

    Exemple 2 : Soit $f(x)=20x^3$.
    Une primitive de $f$ est :
    $F(x) = 5 x^4$.

    Composition :
    Soit $u$ une fonction dérivable et $F$ une primitive de $f$.
    Une primitive de $u'f(u)$ est : $F(u)$.
    $(F (u(x)))' = u'(x) \times f(u(x))$.

    Exemple 3 : Soit $f(x) = 2 \text{e}^{2x-1}$.
    Une primitive de $f$ est :
    $F(x) = \text{e}^{2x-1}$.

    Compositions particulières :
    Une primitive de $n u' u^{n-1}$ est : $u^n$.
    Une primitive de $u' \text{e}^u$ est : $\text{e}^u$.
    Une primitive de $\dfrac{u'}{u}$ est : $\text{ln}(|u|)$.
    Une primitive de $\dfrac{u'}{\sqrt{u}}$ est : $2 \sqrt{u}$.
    Une primitive de $\dfrac{u'}{u^2}$ est : $- \dfrac{1}{u}$.

    Exemple 4 : Soit $f(x) = \dfrac{2x}{x^2+1}$.
    Une primitive de $f$ est :
    $F(x) = \text{ln}(x^2+1)$.
